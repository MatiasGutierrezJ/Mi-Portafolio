Habilidades

Estas son algunas de las tecnologías y conceptos que manejo y utilizo para crear sitios web funcionales, ordenados y adaptables a distintos dispositivos.

  • HTML: Estructuración semántica de páginas web, uso correcto de etiquetas, jerarquía de contenidos, enlaces, listas, secciones y buenas prácticas para una base sólida.
  • CSS: Creación de estilos visuales, manejo de colores, tipografías, espaciados, diseño responsive con media queries y animaciones suaves.
  • Flexbox: Alineación y distribución de elementos en una sola dimensión, centrado de contenido, manejo de espacios y adaptación a diferentes tamaños de pantalla.
  • CSS Grid: Organización de layouts más complejos usando filas y columnas, control del diseño general de la página y estructuras ordenadas y responsivas.
